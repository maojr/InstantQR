# InstantQR
A chrome extension to generate the QR code image of you current tab's url.

InstantQR - 产生当前标签页的 URL 的 QR 图.

![](src/icon48.png)

# API
使用了以下开源项目的代码:

[qrcodejs](https://github.com/davidshimjs/qrcodejs)
